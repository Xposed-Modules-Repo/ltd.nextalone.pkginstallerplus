## Compatibility

Compatible with AOSP-like installer,been tested in:  

1. Hydrogen OS Android 7.1.1
2. Oxygen OS Android Q
3. Pixel Experience 11
4. LineageOS Android 12
5. LineageOS Android 13
6. Pixel/AOSP Android 13
